## 紧急响应阶段（封号后24小时内）
### 立即停损三步法
1. **冻结关联操作**  
   - 断开所有绑定广告账户  
   - 移除关联第三方应用授权  
   - 关闭「寻找好友」定位功能  
2. **环境隔离设置**  
   - 启用全新设备或虚拟机操作  
   - 更换独立公网IP地址（推荐美国静态住宅IP）  
   - 创建临时邮箱用于申诉通讯  
3. **数据快照保存**  
   - 导出过去30天互动记录  
   - 备份主页管理权限截图  
   - 留存广告消费账单PDF
